# Chef-managed, Docker-based ELK stack

DELK is a Docker-based ELK stack (Elasticsearch, Logstash, Kibana)
inside a Vagrant box, provisioned by Chef and Kitchen.


# Implementation

- [x] Elasticsearch
- [x] Logstash
- [ ] Kibana
- [ ] Curator
- [ ] Maintenance scripts
